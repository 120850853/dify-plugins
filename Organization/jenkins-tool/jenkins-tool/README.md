# Jenkins Tool

A Dify plugin for interacting with Jenkins servers, helping you retrieve build information, view build logs, and trigger new builds.

## Features

- Get Jenkins build information
- View build Console-output
- Trigger new builds

## Prerequisites

Before using this plugin, you need:

1. An accessible Jenkins server
2. Jenkins server credentials (username and API Token)

## Configuration

Before using the plugin, you need to configure the following information:

1. Jenkins server address (e.g., `http://your-jenkins-server:8080`)
2. Jenkins username
3. Jenkins API Token (can be generated in Jenkins user configuration page)

## Usage

1. Install this plugin on the Dify platform
2. Configure Jenkins server information
3. Use the provided tools to interact with Jenkins

## Notes

- Ensure the Jenkins server address is accessible
- Keep your API Token secure and do not share it with others
- It is recommended to use HTTPS protocol when accessing Jenkins server
